# 작업 규칙

## 담당 파트
- **내 파트:** `backend/src/main/java/com/example/fitness/community/`
- 이 경로 안의 파일만 수정 가능

## 절대 금지
- `frontend/` 하위 파일 수정 금지
- 다른 백엔드 파트 수정 금지 (`config/`, `entity/`, `repository/` 등 community 외부)

## 다른 파트를 건드려야 할 경우
- 직접 수정하지 않는다
- **이유**와 **수정이 필요한 위치**를 사용자에게 말로 보고하고 허락을 받는다

## 팀 구성
- 이태관: 회원가입/로그인 + 칼로리계산 프론트엔드
- 박주영: 회원가입/로그인 + 칼로리계산 백엔드
- 이승훈: 운동 추천 및 루틴 관리 프론트엔드
- 김재연: 운동 추천 및 루틴 관리 백엔드
- 신현서: 커뮤니티 기능 프론트엔드
- 이재근(본인): 커뮤니티 기능 백엔드
